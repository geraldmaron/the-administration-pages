# The Administration — Public Pages

Static HTML hosted on GitHub Pages for the App Store-required URLs.

- `index.html` — landing page
- `privacy.html` — Privacy Policy (App Store Connect → App Privacy → Privacy Policy URL)
- `terms.html` — Terms of Service
- `support.html` — Support page (App Store Connect → App Information → Support URL)
- `style.css` — shared styles
- `.nojekyll` — disables Jekyll so files are served as-is

## Deployment

Mirrored to a separate public repo (`the-administration-pages`) that hosts GitHub Pages.
The source repo (`administration-app`) is private; Pages requires public for free tier.
